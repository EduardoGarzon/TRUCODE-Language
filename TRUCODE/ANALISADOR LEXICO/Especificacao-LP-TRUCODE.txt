--------------------------------------------------------------------------------------------------------
1. Nome da Linguagem: TRUCODE
--------------------------------------------------------------------------------------------------------

--------------------------------------------------------------------------------------------------------
2. Tipos de dados primitivos suportados: 

    - int     (OURO)   -> Números inteiros.
    - float   (ESPADA) -> Números reais.
    - string  (COPA)   -> Textos.
    - boolean (PAUS)   -> Verdadeiro ou Falso.
--------------------------------------------------------------------------------------------------------

--------------------------------------------------------------------------------------------------------

        - AND   (##)

    C.	Operadores Aritméticos:

--------------------------------------------------------------------------------------------------------

--------------------------------------------------------------------------------------------------------
4. Formação dos Identificadores (Regra para Nomes de Variáveis):

    - Expressão Regular : [a-zA-Z]([a-zA-Z0-9_])*
    - Tamanho Mínimo    : 1 caractere.     
    - Tamanho Máximo    : 12 caracteres.    
    - Não podem ser palavras reservadas.
    - Case Sensitive.   

    EXEMPLOS:
        - Declaração             	: Nome-da-Variavel[Tipo da Variável]#
        - Declaração e Atribuição	: Nome-da-Variavel[Tipo da Variãvel] VALE valor#
--------------------------------------------------------------------------------------------------------
 
--------------------------------------------------------------------------------------------------------
5. Comandos de Entrada e Saída (Um para Entrada e Um para Saída):

    - Entrada   : RECEBE -> RECEBE[identificador]# 
    - Saída     : MOSTRA -> MOSTRA[lista de valores]#

9. Estrutura Geral do Programa (Sintaxe):


-----------------------------------------------
EXEMPLO 1:
-----------------------------------------------
@@ Declaração de Variáveis
limite[OURO] VALE 0#
counter[OURO]#

@@ Entrada e Saída de Dados
MOSTRA["Informe um valor: "]#
RECEBE[limite]#

@@ Estrutura Condicional
TRUCO(limite MATA 0) {

    @@ Laco de Repeticao Contado
    DISTRIBUI counter DE 0 ATE limite PASSO 1 {
        MOSTRA[counter]#
    }
}
CORRE {
    MOSTRA["O limite deve ser maior que 0."]#
}

-----------------------------------------------
EXEMPLO 2:
-----------------------------------------------
numA[OURO]#
numB[OURO]#
numC[ESPADA]#
resultado[ESPADA]#

RECEBE[numA]#
RECEBE[numB]#
RECEBE[numC]#

@@ Expressão Aritmética
resultado VALE ((numA KING numB) JACK (numC QUEEN 2))#

MOSTRA["O resultado vale: ", resultado]#

TRUCO(resultado CANTA 0) {
    MOSTRA["O resultado é maior ou igual a zero."]#
}
CORRE {
    MOSTRA["O resultado é menor que zero."]#
}

resultado VALE XX(((10 BLEFA 10) ## (5 RECUA 10)) ?? (100 MATA 50))#
